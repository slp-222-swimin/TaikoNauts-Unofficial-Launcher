# プレイヤーデータ / ネームプレート編集機能の仕様書

## 1. 基本仕様
- Skins 項目のApply selected skinの右に`Edit`ボタンを追加
- ボタンを押下して編集ウィンドウを開く
- `[選択したディレクトリ]\PlayerData\[n]` にプレイヤーデータが入っている。`n >= 0` で、編集ウィンドウ上部でなんのデータを編集するか選べるようにする
- すべての編集項目において、`Int` なら数字入力ボックス、`String` なら文字入力ボックス、`Boolean` ならチェックボックスを表示する
- `[選択したディレクトリ]\PlayerData\[n]\PlayerConfig.json` 内にプレイヤーデータが格納。形式と説明は以下を参照
```json
{
  "isShowScore": true, // スコアを表示する
  "isSaveScore": true, // スコアを保存する
  "donchanType": 0, // 選択したキャラのタイプ（無視）
  "isUsePuchiChara": false, // ぷちきゃらを使うかどうか（無視）
  "puchiCharaType": 0, // 選択したぷちきゃらのタイプ（無視）
  "songSelectScorePanelIndex": 4 // どの難易度の総合成績パネルを表示するか（0:Easy,1:Normal,2:Hard,3:Oni,4:Oni / Edit）
}
```
- `[選択したディレクトリ]\PlayerData\[n]\NamePlateConfig.json` 内にネームプレートのプロパティが格納。形式と説明は以下を参照
```json
{
  "name": "Sleepy", // プレイヤー名
  "title": "Яeverse Result", // 称号名
  "rank": "十段", // 段位名
  "isRankGold": true, // 金合格かどうか
  "namePlateType": 0, // 称号プレートのタイプ。詳しい仕様は次項目を参照
  "rankType": 1 // 段位の合格レベル（0:銀,1:金,2:虹）
}
```

## 2. ネームプレートの編集UX
- ネームプレート編集では、ユーザーがプレビューで確認できる機能を搭載する。
- ネームプレートの描画では、`[選択したディレクトリ]\Skins\[選択したスキン]\Image\99.Common\NamePlate\NamePlate_Base.png` を前項目JSONに沿ってクリッピングしたものを使用する。
- `[選択したディレクトリ]\PlayerData\[n]\NamePlateConfig.json` の`namePlateType` では、`[選択したディレクトリ]\Skins\[選択したスキン]\Image\99.Common\NamePlate\Plates\[n]\Base.png` を画像として表示。`n = namePlateType , n >= 0`。
