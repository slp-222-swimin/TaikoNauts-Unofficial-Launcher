⚙️ システム詳細設定 (System Config)

「Config」タブでは、ゲーム設定ファイル（Config/SystemConfig.json）の内容をGUIから編集し、反映・保存します。

💾 保存と再読み込み

Save（保存）: 設定項目を変更した後、画面右上の「Save」ボタンをクリックすることで、設定ファイルへ書き込まれます。

Reload（再読み込み）: 「Reload」ボタンをクリックすると、設定ファイルから現在の値を再読み込みして表示を更新します。

📺 画面表示設定 (Display)

解像度、フレームレート、および背景動画の挙動を調整します。

項目名 (UI) JSONキー コントロール形式 詳細説明
Fullscreen fullscreen Checkbox (ON/OFF) フルスクリーン表示の有無を設定します。
V-Sync vSync Checkbox (ON/OFF) 垂直同期の有効/無効を設定します。
Show FPS showFPS Checkbox (ON/OFF) ゲーム中に現在のフレームレート(FPS)を画面上に表示します。
Background Movie useBackgroundMovie Checkbox (ON/OFF) 演奏画面等の背景で動画を再生するかを指定します。
Resolution resolution Dropdown / Pop-up 解像度を設定します。プリセット値のほか、「Custom」を選択すると幅と高さを数値で直接指定するダイアログが開きます。
Target FPS targetFPS Slider (0〜240) 最大フレームレートを制限します。0 指定時は無制限となります。
Movie Type backgroundMovieType Dropdown 背景動画のブレンドモードを指定します（None / Blur / BlurWithMiniWindow）。
Movie Opacity backgroundMovieOpacity Slider (0.00〜1.00) 背景動画の不透明度を設定します。
Movie Blur Amount backgroundMovieBlurAmount Slider (0.00〜1.00) 背景動画のぼかし（Blur）量を設定します。

🎵 オーディオ設定 (Audio)

音声出力APIや各種音量、オフセットを定義します。

項目名 (UI) JSONキー コントロール形式 詳細説明
Audio Mode audioMode Dropdown 音声の出力先APIを選択します（DirectSound / WASAPIShared / WASAPIExclusive / ASIO）。
Audio Buffer Size audioBufferSize Slider (5〜200) オーディオのバッファサイズ（ミリ秒）です。数値を下げるほど遅延は減少しますが、環境によっては音割れの原因となります。
Master Volume masterVolume Slider (0〜100) ゲーム全体の主音量を調整します。
SE Volume seVolume Slider (0〜100) 打鍵音などの効果音音量です。
Music Volume musicVolume Slider (0〜100) 演奏する曲の音量です。
BGM Volume bgmVolume Slider (0〜100) 選曲メニュー画面などで流れるBGMの音量です。
Voice Volume voiceVolume Slider (0〜100) キャラクターボイスの音量です。
Sound Offset overallSoundOffset Numeric Entry (ms) 音ズレ全体の調整値（ミリ秒単位）です。
Note Position Offset overallNotePositionOffset Numeric Entry ノーツが流れる描画位置のオフセット設定です。
Input Delay inputDelay Numeric Entry (ms) キー入力等の入力信号に対する遅延オフセット値（ミリ秒単位）です。

🎮 ゲームプレイ設定 (Gameplay)

ゲーム内のシステムルールや各種演出設定を調整します。

項目名 (UI) JSONキー コントロール形式 詳細説明
Preload Fonts preloadFonts Checkbox (ON/OFF) 起動時にすべてのフォントデータをキャッシュし、演奏開始時の遅延を防止します。
Use Donchan useDonchan Checkbox (ON/OFF) 演奏画面等のどんちゃん表示の有無を設定します。
Clear New Song on Play clearNewSongOnPlay Checkbox (ON/OFF) 一度でも楽曲をプレイした際、選曲画面の「NEW」マークを自動的に消去します。
Stop Animation in Enso animationStopInEnsoGame Checkbox (ON/OFF) 演奏中に各種アニメーションを停止し、動作負荷を低減します。
Judge Frame Afterimage showJudgeFrameAfterimage Checkbox (ON/OFF) 判定枠の残像エフェクト表示の有無を設定します。
New Result Animation useNewResultAnimation Checkbox (ON/OFF) リザルト画面に新しいアニメーション演出を使用するか設定します。
Include Gaiden in Dan includeGaidenInDanAssignmentFolder Checkbox (ON/OFF) 段位道場のフォルダ構成に、外伝のお題を含めるかを設定します。
