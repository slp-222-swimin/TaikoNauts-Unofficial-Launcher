🎨 スキンの切り替え・適用

「Skins」タブでは、ゲームフォルダ内のスキンフォルダを検出し、ゲーム設定へ適用します。

🔍 スキンの自動検出仕様

ランチャーは、ゲームフォルダ直下の Skins/ ディレクトリ配下をスキャンします。

📂 フォルダ構造

各スキンフォルダの中に SkinConfig.json が配置されている必要があります。

ランチャーはこれを再帰的に検索し、設定されているスキン情報をリストに取得します。

SkinConfig.json から読み込まれるパラメータ

JSONキー 説明 未設定時の挙動
skinName スキンの表示名 フォルダ名がそのまま表示名になります。
skinVersion スキンのバージョン 空欄になります。
skinDescription スキンの説明文 空欄になります。

🚀 スキンの適用方法

  • リストからスキンを選択

    リストから任意のスキンを選択します。選択されたスキンの情報は、「Name」「Version」「SkinPath」「Description」の項目で一覧表示されます。

  • 設定の更新

    「Apply selected skin」ボタンをクリックします。完了時に通知が表示されます。

💡 適用プロセスの詳細

スキンが適用されると、ゲームフォルダ内の Config/GameConfig.json にある skinPath の値が、選択したスキンの相対パス（例: Skins/SkinFolderName）に更新されます。

また、ランチャー起動時や「Reload skins」クリック時には、現在 Config/GameConfig.json に指定されているスキンが自動的にリスト上で選択されます。

🔄 スキンの手動更新 (Reload)

Skins/ フォルダの内容を手動で変更した場合は、「Reload skins」ボタンをクリックすることで、一覧を最新の状態に更新できます。
