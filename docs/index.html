TaikøNauts UNOFFL Launcher 仕様書

本ツールは、ゲームの起動、アップデーターの制御、スキン切り替え、および譜面ZIPファイルの自動インポートを行うWindows向けのランチャーです。

ℹ️ 概要

Python 3.10 以上で動作し、標準ライブラリおよび Windows API を用いて以下の管理機能を提供します。

🚀 提供機能

🎮 ゲーム起動・インストール

TaikoNauts.exe を起動します。未インストール時は、指定URLからファイルをダウンロードして導入します。

🔄 アップデート管理

TaikoNautsUpdater.exe をコンソール上で実行・監視し、プロンプトへの自動応答を行います。

🎨 スキン切り替え

ゲームフォルダ内の Skins/ からスキン設定を自動検出し、一覧表示。ワンクリックで GameConfig.json に適用します。

📦 譜面ZIP簡単インポート

譜面の .zip ファイルを画面にドラッグ＆ドロップするだけで、Songs/zip へ自動で安全に展開します。

⚙️ システム詳細設定

ゲーム内の詳細な環境設定ファイル SystemConfig.json を、GUI上のグラフィカルなコントローラーから一括編集・保存できます。

💻 動作環境

項目 要件 備考
対応OS Windows 10 / 11 Windows API (Windows API フックなど) を使用しているため、Windows専用です。
Python環境 Python 3.10 以上 ビルド前のソースコードから直接実行する場合に必要です。
追加パッケージ 不要 標準ライブラリおよび Windows API のみで動作するように設計されています。

🏁 クイックスタート

  • ゲームフォルダの設定

    「Game」タブにて、TaikoNauts.exe が配置されているフォルダを「Browse...」ボタンから選択します。設定は launcher_state.json に自動で保存されます。

  • ゲームのインストール（未所持の場合）

    フォルダを選択した際、対象フォルダ内に TaikoNauts.exe が見つからない場合は、ボタンが「Install Game」に切り替わります。これをクリックすると、最新ビルドが自動ダウンロードされます。

  • ゲームの起動と設定

    「Launch Game」をクリックしてゲームを起動するか、各タブでスキンや譜面、システム詳細設定をカスタマイズしてください。
