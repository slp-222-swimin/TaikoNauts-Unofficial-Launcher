🎮 ゲームの起動とインストール・更新

「Game」タブでは、ゲームの起動、新規インストール、およびアップデーターの実行制御を行います。

▶️ ゲームの起動 (Launch Game)

「▶ Launch Game」ボタンを押すと、バックグラウンドで TaikoNauts.exe が起動します。

📥 自動ゲームインストール機能

設定したゲームフォルダに TaikoNauts.exe が存在しない場合、ボタン表示が「⬇ Install Game」に変更されます。

📦 インストールの流れ

1. 「⬇ Install Game」をクリックすると、ゲームの最小構成（アップデーター本体を含む bootstrap ZIP）を指定フォルダにダウンロード・解凍します。

2. 展開後、ランチャーは自動的にこのアップデーター（TaikoNautsUpdater.exe）を起動します。

3. 起動したアップデーターが実行されることで、ゲーム本体のファイル群がダウンロードおよびインストールされます。

🔄 アップデーターの連携・制御 (Launch Updater)

「⬆ Launch Updater」ボタンをクリックすると、同梱されているアップデーター TaikoNautsUpdater.exe を非同期プロセスとして実行し、出力をログ監視します。

1. 事前設定 (Updater Configuration)

アップデーター起動時に設定ダイアログが開き、以下の対話プロンプトに対する自動応答ルールを指定できます。

設定オプション デフォルト値 説明
Overwrite existing "TaikoNauts.exe" ON (True) TaikoNauts.exe を上書きするか尋ねられた際、自動で y (Yes) または n (No) と応答します。
Overwrite "SimpleStyle" skin files ON (True) 標準スキンファイルを上書きするか尋ねられた際、自動で y (Yes) または n (No) と応答します。

2. ログコンソール機能

アップデーターのコンソール出力を読み込み、ランチャー内のリストボックスへ出力します。

  • 自動スクロール: ログが出力されると、自動で最下部までスクロールします。
  • 行数制限: ログバッファは最大 1,000 行に制限されており、これを超えると古いログが自動削除されます。
  • 停止機能 (Stop): アップデーターのプロセスを途中で終了（Terminate）させます。
⚠️ アップデート時の注意点

アップデーターは処理が終了した際、自動でゲーム本体を起動することがあります。ランチャーはこれを防ぎ、安全にアップデートプロセスを終了させるため、アップデーター終了後の20秒間バックグラウンドでゲーム本体の起動を監視し、自動起動したゲームプロセスを検知した場合は直ちに停止（Kill）させる保護機能（_watch_and_kill_game_launch）を実装しています。
